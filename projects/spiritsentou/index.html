Spirit Sentou

Lua
An MMO style fighting game made as a passion project with a group of friends.

About

One of the first projects I seriously worked on. All scripting and animating was done by me, while the modeling and building were done by my friends.

Features

  • Basic combat (Blocking, Parrying, Dodging, M1, M2, Skills)
  • Advanced character customization (w/ 3D clothing)
  • Notifications via Discord Integration
  • Simple datastore
